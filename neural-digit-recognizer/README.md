## Quick Start

### 1. Train the model (one-time, ~2вЂ“5 minutes)

```bash
# From the project root
pip install tensorflow numpy pillow
python train_model.py
```

This downloads MNIST, trains a CNN (~99% test accuracy), and saves
`backend/model/mnist_cnn.h5`.

---

### 2. Start the backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

API is live at http://localhost:8000  
Interactive docs at http://localhost:8000/docs

---

### 3. Start the frontend

```bash
cd frontend
npm install
npm start
```

Opens at http://localhost:3000

## API Reference

### `POST /predict`

**Request body**
```json
{ "image": "<base64-encoded PNG>" }
```

**Response**
```json
{
  "digit": 7,
  "confidence": 0.9981,
  "probabilities": [0.0001, 0.0002, 0.0003, 0.0001, 0.0001, 0.0001, 0.0001, 0.9981, 0.0005, 0.0004]
}
```

### `GET /health`
```json
{ "model_loaded": true }
```
